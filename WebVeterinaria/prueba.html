Acerca de


Este es un proyecto para el uso web, el objetivo es globalizar, unificar, facilitar la información acerca de las mascotas, de esta forma se simplifica el trabajo a la hora de asistir a una veterinaria.
Al ser utilizado desde una web se facilita el acceso y esto hace que el uso sea simple y compatible para cualquier dispositivo.
Para simplificar la búsqueda los dueños o clientes van a ingresar su DNI y esto va a devolver las mascotas registradas que el cliente tiene.
El registro de la mascota tiene su historia médica, estudios realizados, recordatorios.
Cada profesional veterinario y cliente va a tener su interfaz, la que va a proporcionar toda la información referida a sus mascotas. La web también va a contar con diferentes servicios como:
- Ubicaciones de veterinarias con su información
- Veterinarias 24hs
- Contactos para diferentes estudios como radiografias, etc
- Lavado, cortes de pelo, fisioterapeutas
- Refugios
- Donaciones
- Diferentes especialistas
- Busqueda de medicacion:

El cliente y el profesional veterinario podra realizar la busqueda del medicamento lo cual lanzara una alerta o email a las veterinarias, farmacias para facilitar la obtencion del mismo.
